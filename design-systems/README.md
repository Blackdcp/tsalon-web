# T Salon — Visual System Directions

This folder contains three code-native visual-system studies. Each direction includes:

- a complete interactive specimen page;
- a `DESIGN.md` implementation reference;
- a foundation preview;
- a homepage application preview;
- the existing T Salon Logo and real community photographs.

## Directions

1. **Protocol** — precise, open and composable developer-community infrastructure.
2. **Night Build** — a dark editorial workbench for ideas under construction.
3. **Commons** — a warm, technically rigorous community commons.

Open `index.html` to compare them. These are selection studies, not the production website implementation.
